# מטרו ישראל - Metro Israel Website

אתר אינטרנט בעברית לפרויקט המטרו החדש בישראל.

## התקנה והרצה

```bash
# התקנת תלויות
npm install

# הרצת שרת פיתוח
npm run dev

# בניית גרסת ייצור
npm run build

# תצוגה מקדימה של גרסת ייצור
npm run preview
```

## טכנולוגיות

- Vite - כלי בנייה מהיר
- React - ספריית UI
- CSS3 - עיצוב מודרני ורספונסיבי
